# **App Name**: UBEC Data Form

## Core Features:

- User Sign-Up: Allow users to create accounts with email and password.
- User Type Selection: Prompt users to select their type (Individual or Group) after signing up.
- Conditional Form Display: Display relevant form sections based on the selected user type.
- Individual Form: A comprehensive data collection form for individual users.
- Group Form: A tailored data collection form with specific sections for group users.

## Style Guidelines:

- Primary color: Dark green (#2F7438) as per the specified brand color.
- Background: White (#FFFFFF) for a clean and minimal appearance.
- Accent: Light green (#A9D18E), a harmonious analogous color, for highlighting interactive elements.
- Font: 'Montserrat' (sans-serif) as per the design specification. Note: currently only Google Fonts are supported.
- Clean and minimal layout with clear visual hierarchy, guided by provided design files. Form elements should be well-spaced and aligned.
- Simple and intuitive icons where applicable, complementing the minimal design.
